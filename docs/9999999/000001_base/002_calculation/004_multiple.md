
## 解説

JavaScriptの中では、四則演算ができます。

掛け算を行う時は、「*」を使用します。

```javascript
let a = 10;
let b = 2;
let c = 6 * 3;
let d = a * 5;   // -> 50
let e = a * d;   // -> 500
```

### 累乗する

2の3乗など、累乗を行う場合は「^」を使用します。

```javascript
let a = 10;
let b = 2;
let c = 2 ^ 3;   // -> 8
let d = a ^ 3;   // -> 100
```
